# 01 - Basic Examples

This folder contains the simplest possible SongCode examples to help you get started.

## What You'll Learn

- Basic file structure (metadata, sections, lyrics)
- Simple chord progressions without patterns
- Inline chord notation
- Lyric timing with `_n` measure counts

## Examples

### `simple-verse-chorus.sc`
A minimal song with:
- 2 metadata fields
- 2 sections (Verse, Chorus)
- Inline chords (no pattern definitions)
- Simple measure counting

**Demonstrates**:
- File structure basics
- How to write chords inline
- How to time lyrics to measures
- Measure separator `;`

### `minimal-song.sc`
The absolute minimum valid SongCode file:
- No metadata
- 1 section
- 1 chord progression
- No lyrics

**Demonstrates**:
- What's required vs optional
- How to create an instrumental

## Tips for Beginners

1. **Start here**: These examples have no advanced features
2. **Copy-paste**: Use these as templates for your first songs
3. **Experiment**: Change chords, add lyrics, modify measure counts
4. **Next step**: Once comfortable, move to `02-intermediate/` to learn patterns

## Common Patterns in This Folder

- **I-IV-V-I**: `G;C;D;G` (classic progression)
- **I-V-vi-IV**: `C;G;Am;F` (pop progression)
- **Simple timing**: Most lyrics are `_2` (2 measures)
