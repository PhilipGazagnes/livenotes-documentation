# LiveNotes Documentation

Welcome to the **LiveNotes** ecosystem documentation. LiveNotes is a comprehensive platform designed for musicians to store, organize, and access song information in powerful and intuitive ways.

## What is LiveNotes?

LiveNotes is an application ecosystem built specifically for musicians who need to:

- **Store** song information (structure, lyrics, chords)
- **Organize** their repertoire with rich metadata
- **Perform** with a live performance prompter
- **Edit** songs quickly during rehearsals
- **Browse** their song directory with rich previews

## Architecture

The LiveNotes ecosystem consists of:

- **SongCode** - The notation language
- **SongCode Parser** - Converts SongCode to JSON
- **LiveNotes Apps** - Various applications that consume the JSON format
